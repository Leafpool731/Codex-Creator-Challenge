# Experimental portrait editing

This folder stores unfinished AI segmentation, masked recolor, debug-mask, and
overlay work that is not part of the active ShadeSeason MVP.

The production app currently focuses on:

- photorealistic model selection
- skin depth, undertone, saturation, and contrast
- subtle skin and lighting preview overlays
- 16-season scoring and cosmetic palette results

Keep experimental files here until the AI/mask pipeline is ready to reintroduce
as a polished feature.
